## 1. ОСНОВЫ ПРОЧНОСТИ

### Напряжения и деформации
- **Нормальное напряжение**: σ = N/A (растяжение/сжатие)
- **Касательное напряжение**: τ = Q/A (срез)
- **Напряжение кручения**: τ = M_k·r / J_p, для вала: τ = 16·M_k / (π·d³)
- **Напряжение изгиба**: σ = M·y / I, для балки: σ = M / W
- **Закон Гука**: σ = E·ε; τ = G·γ
- **Связь E и G**: G = E / [2(1+μ)], μ≈0.3 для стали, G≈8·10⁴ МПа

### Моменты сопротивления и инерции
| Сечение | W_x | J_x | Примечание |
|---------|-----|-----|------------|
| Круг d | πd³/32 ≈ 0.1d³ | πd⁴/64 ≈ 0.05d⁴ | — |
| Кольцо (труба) | πD³(1-α⁴)/32 | πD⁴(1-α⁴)/64 | α=d/D |
| Прямоугольник b×h | bh²/6 | bh³/12 | — |
| Квадрат a | a³/6 | a⁴/12 | — |
| Швеллер № | По сортаменту | По сортаменту | — |
| Двутавр № | По сортаменту | По сортаменту | — |

### Условия прочности
- **По нормальным напряжениям**: σ_max ≤ [σ] = σ_пред / n
- **По касательным напряжениям**: τ_max ≤ [τ] = τ_пред / n
- **Сложное напряжённое состояние**:
  - III теория (Tresca): σ_экв = σ₁ - σ₃ ≤ [σ]
  - IV теория (Mises): σ_экв = √(σ₁²+σ₂²+σ₃² - σ₁σ₂ - σ₂σ₃ - σ₃σ₁) ≤ [σ]
  - Для плоского НС: σ_экв = √(σ² + 3τ²) ≤ [σ]

### Коэффициенты запаса прочности
| Условия работы | Пластичный материал | Хрупкий материал |
|----------------|--------------------|------------------|
| Точный расчёт, спокойная нагрузка | 1.2-1.5 | 2.0-3.0 |
| Приближённый расчёт | 1.5-2.0 | 3.0-4.0 |
| Ударная нагрузка, высокая ответственность | 2.5-4.0 | 4.0-6.0 |
| Авиационная техника | 1.2-1.8 | 2.0-3.5 |
| Строительные металлоконструкции | 1.3-1.6 | 2.5-3.5 |

## 2. УСТАЛОСТНАЯ ПРОЧНОСТЬ (Анализ по Shigley-Mischke)

### Предел выносливости
Для стали: σ'_e = 0.504·σ_uts при σ_uts ≤ 1400 МПа
Для стали: σ'_e = 700 МПа при σ_uts ≥ 1400 МПа
Для чугуна: σ'_e = 0.4·σ_uts
Для алюминия: σ'_e = 0.35·σ_uts

### Факторы, снижающие предел выносливости (Marin)
σ_e = k_a · k_b · k_c · k_d · k_e · k_f · σ'_e

| Фактор | Формула | Примечание |
|--------|---------|------------|
| k_a — качество поверхности | k_a = a·σ_uts^b | Полировка: a=1.58, b=-0.085; Шлифовка: a=1.34, b=-0.085; Токарная: a=4.51, b=-0.265; Грубая: a=57.7, b=-0.718 |
| k_b — размер | k_b = (d/7.62)^(-0.1133) для 2.79≤d≤51 мм; k_b = 1.51·d^(-0.157) для 51<d≤254 мм | Для круглого сечения |
| k_c — надёжность | k_c = 1-0.08·z_a | 50%:1.0; 90%:0.897; 95%:0.868; 99%:0.814; 99.9%:0.753; 99.99%:0.702 |
| k_d — температура | k_d = 1 при T≤450°C; k_d = 1-0.0032·(T-450) при T>450°C | Для стали |
| k_e — нагрузка | Изгиб:1.0; Осевая:0.85; Кручение:0.59 | — |
| k_f — прочие | k_f = 1 — для базового расчёта | — |

### Концентрация напряжений
- **Теоретический K_t**: Находится по графикам/таблицам для данного типа концентратора (галтель, отверстие, канавка, шпоночный паз)
- **Коэффициент чувствительности к надрезу**: q = (K_f - 1)/(K_t - 1)
  - Для стали с σ_uts < 560 МПа: q = 0.02-0.3 (малочувствительна)
  - Для стали с σ_uts > 1400 МПа: q = 0.6-0.9 (высокочувствительна)
- **Коэффициент снижения выносливости**: K_f = 1 + q·(K_t - 1)
  - Для шпоночного паза (фреза): K_f ≈ 1.6-2.0
  - Для резьбы: K_f ≈ 2.0-3.5
  - Для галтели d/D=1.2, r/d=0.1: K_f ≈ 1.4-1.8

### Критерии усталостного разрушения
| Критерий (Soderberg, Goodman, Gerber, ASME) | Уравнение |
|----------------------------------------------|-----------|
| **Soderberg** (консервативный) | σ_a/S_e + σ_m/S_y = 1/n |
| **Goodman (модиф.)** (распространён) | σ_a/S_e + σ_m/S_uts = 1/n |
| **Gerber** (параболический) | n·σ_a/S_e + (n·σ_m/S_uts)² = 1 |
| **ASME** (для кручения+изгиба) | σ_a = k·M_b·c/I, τ_m = T·r/J; σ_экв = √(σ_a² + 3·(τ_m·S_e/S_y)²) |

### Расчёт вала на усталость (по ASME B106.1M-1985)
d = [ (32·n_s/π) · √( (M/σ_e)² + 0.75·(T/σ_y)² ) ]^(1/3)
Где: d — диаметр вала (м), n_s — запас прочности, M — изгибающий момент (Н·м),
T — крутящий момент (Н·м), σ_e — предел выносливости детали (Па), σ_y — предел текучести (Па)

## 3. РАСЧЁТ СОЕДИНЕНИЙ

### Сварные швы (статическая нагрузка)
| Тип шва | Условие прочности | Коэф. прочности шва φ |
|---------|-------------------|----------------------|
| Стыковой (ручная сварка) | σ' = F/(l·s) ≤ [σ_р]·φ | φ = 0.65-0.85 (ручная) |
| Стыковой (автоматическая) | σ' = F/(l·s) ≤ [σ_р]·φ | φ = 0.90-0.95 (автомат.) |
| Угловой лобовой | τ' = F/(0.7·k·l) ≤ [τ'_св] | — |
| Угловой фланговый | τ' = F/(0.7·k·Σl) ≤ [τ'_св] | — |
| Тавровый (без разделки) | τ' = F/(0.7·k·l) ≤ [τ'_св] | — |
| Тавровый (с разделкой) | σ' = F/(l·s) ≤ [σ_р]·φ | — |

- k — катет шва (обычно k ≈ 0.8·s, где s — толщина тонкой детали)
- [τ'_св] = 0.6·[σ_р] для электродов Э42 (Ст3)
- [τ'_св] = 0.65·[σ_р] для электродов Э50 (сталь 20)
- Минимальный катет шва: k_min = 4 мм (s=4-16 мм), k_min = 6 мм (s=17-22 мм)

### Резьбовые соединения (статическая нагрузка)
- **Внутренний диаметр резьбы**: d₁ = d - 0.9382·P (для метрической резьбы)

  - M24: d=24, P=3.0, d₁=20.752 мм

- **Напряжение в стержне болта**: σ_р = 4·F / (π·d₁²)
- **Момент затяжки**: M_зат = 0.2·F_зат·d (ориентировочно)

### Шпоночные соединения (ГОСТ 23360-78)
- **Напряжение смятия**: σ_см = 2·T / (d·b·(h-t₁)·l_p) ≤ [σ_см]
  - [σ_см] = 60-100 МПа (чугун), 100-150 МПа (сталь, спокойная нагрузка)
  - l_p = l - b (рабочая длина шпонки со скруглёнными торцами)
- **Шпонка на срез**: τ_ср = 2·T / (d·b·l_p) ≤ [τ_ср] ≈ 70-100 МПа

### Заклёпочные соединения
- **На срез**: τ = 4·F / (π·d²·z·i) ≤ [τ_ср]
- **На смятие**: σ_см = F / (d·s_min·z) ≤ [σ_см]
- **На растяжение по ослабленному сечению**: σ = F / ((b - z·d)·s_min) ≤ [σ]
  - z — число заклёпок, i — число плоскостей среза

## 4. ПРУЖИНЫ

### Винтовые цилиндрические пружины сжатия/растяжения
- **Индекс пружины**: C = D/d (рекомендуется C = 4-12)
- **Коэффициент (Wahl)**: k_w = (4C-1)/(4C-4) + 0.615/C
- **Напряжение**: τ = 8·k_w·F·D / (π·d³) ≤ [τ]
- **Жёсткость**: k = G·d⁴ / (8·D³·n) — сила на единицу сжатия
- **Число рабочих витков**: n = G·d⁴ / (8·D³·k)
- **Полное число витков**: n₁ = n + 1.5..2 (поджатые витки)
- **Длина пружины в свободном состоянии**: L₀ = n·d + n₁·s + (n₁-n)·d, где s = зазор между витками
- **Критическая длина сжатия**: L_cr = n₁·d (до соприкосновения витков)
- **Усилие при полном сжатии**: F_max = k·(L₀ - L_cr)

## 5. ПОДШИПНИКИ КАЧЕНИЯ

### Долговечность (ISO 281, ГОСТ 18855)
- **Номинальная долговечность**: L₁₀ = (C/P)^p · 10⁶ оборотов
  - p = 3 (шариковые)
  - p = 10/3 (роликовые)
- **Долговечность в часах**: L₁₀h = (10⁶/(60·n)) · (C/P)^p
- **Эквивалентная динамическая нагрузка**:
  - Радиальные шариковые: P = X·F_r + Y·F_a
  - Радиальные роликовые: P = F_r (при F_a/F_r ≤ e)
- **Коэффициенты X, Y**: по таблицам в зависимости от F_a/C₀ и типа подшипника
