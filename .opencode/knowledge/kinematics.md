## 1. ТЕОРИЯ МЕХАНИЗМОВ И МАШИН (ТММ)

### Основные понятия
- **Механизм** — система тел, преобразующая движение одного/нескольких тел в требуемое движение
- **Звено** — твёрдое тело (или группа тел с неизменным взаимным расположением)
- **Кинематическая пара** — подвижное соединение двух звеньев
- **Степень свободы кинематической пары**: H = 6 - S (S — число связей)
- **Кинематическая цепь** — система звеньев, соединённых кинематическими парами

### Классификация кинематических пар (по Артоболевскому)
| Класс | Связей S | Подвижность H | Название | Пример |
|-------|----------|--------------|----------|--------|
| I | 1 | 5 | Шар-плоскость | Шар на плоскости |
| II | 2 | 4 | Цилиндр-плоскость | Цилиндр на плоскости |
| III | 3 | 3 | Сферическая | Шаровой шарнир |
| IV | 4 | 2 | Цилиндрическая | Цилиндр-цилиндр |
| V | 5 | 1 | Вращательная/поступат. | Петля, ползун |

### Формула Чебышева (плоские механизмы)
W = 3·n - 2·p₅ - p₄
Где: W — степень свободы, n — число подвижных звеньев,
p₅ — число пар V класса, p₄ — число пар IV класса

**Требование работоспособности**: W ≥ 1 (для механизма с одним ведущим звеном W=1)

### Структурные группы Ассура
- **Группа II класса 1 вида**: 2 звена, 3 вращательных пары (кривошип-шатун-ползун)
- **Группа II класса 2 вида**: 2 звена, 2 вращ.+1 поступат.
- **Группа II класса 3 вида**: 2 звена, 1 вращ.+2 поступат.
- **Группа III класса**: 4 звена, 6 кинематических пар

## 2. РЫЧАЖНЫЕ МЕХАНИЗМЫ

### Кривошипно-шатунный механизм (КШМ)
Параметры: R — радиус кривошипа, L — длина шатуна, λ = R/L (0.15-0.3)
- Ход ползуна: S = 2·R
- Перемещение: x(φ) = R·(1 - cosφ) + (λ·R/4)·(1 - cos2φ)
- Скорость: v(φ) = ω·R·(sinφ + (λ/2)·sin2φ)
- Ускорение: a(φ) = ω²·R·(cosφ + λ·cos2φ)
- Угол давления γ: max при φ=90°, tgγ_max = λ

### Кулисный механизм
- Коэффициент изменения скорости: K = (180+θ)/(180-θ)
- θ = 2·arcsin(R/L) — угол перекрытия
- При K>1 — быстрый обратный ход

### Шарнирный четырёхзвенник
- **Условие проворачиваемости (Грассофа)**: l_min + l_max ≤ l₁ + l₂
- Если min и max — смежные: кривошипно-коромысловый механизм
- Если min — стойка: двухкривошипный
- Если min и max — противоположные: двухкоромысловый

## 3. ЗУБЧАТЫЕ ПЕРЕДАЧИ

### Основная теорема зацепления (Эйлера-Савари)
Общая нормаль в точке контакта делит межосевое расстояние на части,
обратно пропорциональные угловым скоростям: i₁₂ = ω₁/ω₂ = O₂P/O₁P

### Геометрия эвольвентного зацепления (α=20°)
- Радиус основной окружности: r_b = r·cosα = (m·z/2)·cos20°
- Эвольвентная функция: invα = tanα - α (где α — в радианах)
- Угол зацепления в передаче: α_w из invα_w = invα + 2·(x₁+x₂)·tanα/(z₁+z₂)
- Коэффициент суммы смещений: x_Σ = x₁ + x₂

### Расчёт на контактную прочность (ISO 6336-2:2019, ГОСТ 21354-87)
**Контактное напряжение в полюсе зацепления:**
σ_H = Z_E · Z_H · Z_ε · √(F_t · K_H / (b · d₁) · (u+1)/u) ≤ σ_HP

Где:
- Z_E = 189.8 МПа^½ (для стальных колёс) — коэффициент упругости материалов
- Z_H = 2.5 (для α=20°, x_Σ=0) — коэффициент формы (табличный)
- Z_ε — коэффициент перекрытия (≈0.85-0.92 для прямозубых)
- F_t = 2·T₁/d₁ — окружная сила (Н)
- b — ширина венца (мм)
- d₁ — делительный диаметр шестерни (мм)
- u = z₂/z₁ — передаточное число (u ≥ 1)
- K_H = K_A · K_V · K_Hβ · K_Hα — коэффициент нагрузки
- σ_HP — допускаемое контактное напряжение

**Допускаемые контактные напряжения:**
σ_HP = σ_Hlim · Z_N · Z_R · Z_v · Z_W · Z_X / S_Hmin

Где: σ_Hlim — предел контактной выносливости (табличный)
- Для стали: σ_Hlim = 2·HB + 70 (МПа) — при нормализации/улучшении
- Для цементованных: σ_Hlim = 1500-1600 МПа
- Для азотированных: σ_Hlim = 1100-1300 МПа
- Z_N — коэф. долговечности
- S_Hmin ≥ 1.1-1.3 (запас по контактной прочности)

### Расчёт на изгибную прочность (ISO 6336-3:2019, ГОСТ 21354-87)
σ_F = F_t · K_F / (b · m) · Y_Fa · Y_Sa · Y_ε · Y_β ≤ σ_FP

Где:
- Y_Fa — коэффициент формы зуба (зависит от z и x — смещения)
  - Для z=17, x=0: Y_Fa ≈ 2.95
  - Для z=30, x=0: Y_Fa ≈ 2.53
  - Для z=50, x=0: Y_Fa ≈ 2.32
  - Для z=100, x=0: Y_Fa ≈ 2.18
- Y_Sa — коэффициент концентрации напряжений у ножки зуба (≈1.4-1.8)
- Y_β — коэффициент угла наклона зуба
- m — модуль (мм)
- σ_FP — допускаемое напряжение изгиба

**Допускаемые напряжения изгиба:**
σ_FP = σ_Flim · Y_ST · Y_NT · Y_δrelT · Y_RrelT · Y_X / S_Fmin

Где: σ_Flim ≈ 0.4·HB + 100 (для стали 45 улучшение)
S_Fmin ≥ 1.4-1.7 (запас по изгибу)

## 4. ЧЕРВЯЧНЫЕ ПЕРЕДАЧИ

### Параметры
- Коэффициент диаметра червяка: q = d₁/m (ГОСТ 2144-93: q = 8; 10; 12.5; 16; 20)
- Угол подъёма витка: γ = arctg(z₁/q)
- d₁ = q·m; d₂ = m·z₂
- Межосевое: a_w = m·(q+z₂+2·x₂)/2
- КПД: η = tgγ / tg(γ+ρ′), где ρ′ = arctg(f′) — привед. угол трения
  - Для бронзовой пары: f′ = 0.04-0.08 (γ от 5° до 25°)
  - Для чугунной пары: f′ = 0.06-0.12

## 5. ПЛАНЕТАРНЫЕ ПЕРЕДАЧИ

### Схема 2K-H (Джемса)
- i_H = 1 + z_c/z_a (водило неподвижно)
- i_ab_H = -z_b/z_a (отношение при остановленном водиле)
- Условие соосности: z_a + 2·z_c = z_b
- Условие соседства: (z_a+z_c)·sin(π/k) > z_c + 2 (k — число сателлитов)
- Условие сборки: (z_a+z_b)/k = целое число

### Схема 2K-H с двумя внешними (дифференциал)
- i_ab_H = -z_b/z_a
- i_aH_b = 1 - i_ab_H = 1 + z_b/z_a
- Условие соосности: z_b = z_a + 2·z_c

### КПД планетарных передач
η_пл = 1 - (1-η_об) · (i_H - 1)/i_H
Где η_об — КПД обычной передачи (для 1 ступени η_об = 0.97-0.99)

## 6. ВОЛНОВЫЕ ПЕРЕДАЧИ

### Параметры
- i = -z_b / (z_b - z_d) ≈ -z_b / 2, где z_b — колёсо с внутр. зубьями
- z_b - z_d = 2 (разность зубьев, типовое значение)
- Обычно i = 80-320 (в одной ступени)
- КПД: η = 0.75-0.90 (зависит от i и нагрузки)
- Деформация гибкого колеса: w₀ = (0.5-1.0)·m

### Расчёт клиноремённой передачи
- Мощность на 1 ремень: P₁ = (P₀ · C_α · C_L · C_p) / C_i
  - P₀ — баз. мощность на 1 ремень (табличная)
  - C_α — коэф. угла обхвата
  - C_L — коэф. длины ремня
  - C_p — коэф. режима работы
- Число ремней: z = P_расч / P₁
- Угол обхвата малого шкива: α₁ = 180° - 57°·(D₂-D₁)/a ≥ 120°

### Расчёт цепной передачи
- Шаг цепи: p ≈ (25.4·(T·K_э) / (z₁·[p]))^⅓
- I — крутящий момент на малой звёздочке (Н·м)
- K_э = K_д · K_а · K_н · K_рег · K_с · K_см (коэф. эксплуатации)
- Число звеньев цепи: L_p = 2·a/p + (z₁+z₂)/2 + ((z₂-z₁)/(2·π))² · p/a
- Делительный диаметр звёздочки: d = p / sin(180°/z)
- Коэф. запаса прочности: n = F_разр / F_расч ≥ [n] = 7-12
