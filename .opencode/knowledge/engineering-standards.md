## 7. ЗУБЧАТЫЕ КОЛЁСА

### Модули (ГОСТ 9563-60)
| Тип ряда | Модули m, мм |
|----------|-------------|
| 1-й (предпочт.) | 1; 1.25; 1.5; 2; 2.5; 3; 4; 5; 6; 8; 10; 12; 16; 20; 25 |
| 2-й ряд | 1.125; 1.375; 1.75; 2.25; 2.75; 3.5; 4.5; 5.5; 7; 9; 11; 14; 18; 22 |

### Параметры цилиндрической прямозубой передачи
- Делительный диаметр: d = m·z
- Диаметр вершин: d_a = m·(z+2)
- Диаметр впадин: d_f = m·(z-2.5)
- Межосевое расстояние: a_w = m·(z₁+z₂)/2 = (d₁+d₂)/2
- Минимальное число зубьев: z_min = 17 (без подрезания, α=20°)
- Коэффициент перекрытия: ε_α = √(d_a₁²-d_b₁²) + √(d_a₂²-d_b₂²) - a_w·sinα / (π·m·cosα)
- Требование: ε_α ≥ 1.2 (оптимально ε_α = 1.3-1.6)

### Параметры косозубой передачи
- Угол наклона: β = 8-22° (рекомендуется 10-16°)
- Нормальный модуль: m_n (стандартный)
- Торцевой модуль: m_t = m_n / cosβ
- Делительный диаметр: d = m_n·z / cosβ = m_t·z
- Минимальное число зубьев: z_min = 17·cos³β
- Осевое перекрытие: ε_β = b·tanβ / (π·m_n)
- ε_γ = ε_α + ε_β (суммарное перекрытие)

### Параметры конических колёс
- Передаточное число: i = z₂/z₁ = sinδ₂/sinδ₁
- Ортогональная передача (90°): δ₁ + δ₂ = 90°
- Внешний делительный диаметр: d_e = m_te·z (m_te — торцевой модуль)
- Внешнее конусное расстояние: R_e = 0.5·m_te·√(z₁²+z₂²)
