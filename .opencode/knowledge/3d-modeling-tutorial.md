## 1. Философия 3D-моделирования в nanoCAD

> **Источник:** [nanoCAD 3D Solid Modeling Features](https://nanocad.com/products/nanocad-3d-modeling-module/features/)

3D Solid Modeling Module расширяет платформу nanoCAD тремя режимами
моделирования и инструментами сборки:

- **Parametric 3D solid modeling (параметрическое)** — инженеры строят 3D-геометрию
  «кусок за куском». 2D-эскизы превращаются в 3D-фичи (features). Конструктор
  документирует «замысел» через ограничения (constraints) и геометрические связи.
  Каждый шаг следует за серией предыдущих, поэтому параметрический подход
  требует тщательного предварительного планирования. История построения модели
  записывается в **history tree** — в nanoCAD это панель **Inspector** со своими
  инструментами.

- **Direct 3D modeling (прямое)** — подход «что видишь, то и получаешь» (WYSIWYG).
  Дизайнеры редактируют 3D-модели путём **push and pull faces and edges**
  (вытягивания граней и рёбер). Главное преимущество — лёгкость изменения
  дизайна, быстрая итерация и прототипирование.

- **Natural 3D modeling UI (естественный интерфейс)** — nanoCAD отображает команды
  группами для каждого типа моделирования: параметрическое, прямое, листовое
  (sheet metal), каркасное (meshing). Например, в параметрическом режиме
  отображаются команды для создания 3D-геометрии на основе параметрических
  эскизов; при переключении на mesh-моделирование становятся доступны команды
  для создания 3D-поверхностей. Команды, применимые ко всем формам моделирования,
  всегда отображаются на ленте: **chamfers, rounds, Boolean operations, construction geometry**.

Дополнительные возможности:

- **3D constraints** — наложение зависимостей на геометрию, чаще всего для сборки
  деталей в единую модель. Пять типов: **Mate, Insert, Angle, Tangent, Symmetry**.
  Ограничения для каждой детали отображаются в **3D History Tree**.
- **Associativity of 3D models and drawings** — ассоциативная связь моделей и их
  проекций позволяет сосредоточиться на 3D-дизайне, обращая внимание на 2D-графику
  в основном для оформления чертежа. Все изменения 3D-модели в любой момент
  отображаются на чертежах, и элементы дизайна, сделанные в 2D-графике, также
  соответствуют этим изменениям.
- **Bounding prisms (ограничивающие призмы)** — nanoCAD использует 3D-отсечение,
  чтобы показать внутренности моделей и изолировать части. Каждое окно
  пространства модели может содержать **bounding prism** — объём в 3D-пространстве,
  за пределами которого элементы чертежа не отображаются.
- **External references (внешние ссылки)** — nanoCAD связывает внешние ссылки внутри
  сборочных чертежей с деталями, хранящимися во внешних файлах чертежей. Изменения,
  внесённые другими программами дизайна во внешние ссылки, отслеживаются
  и сообщаются через bubble-уведомления.
- **3D navigation tools** — интерактивные инструменты 3D-навигации для отображения
  и навигации вокруг 3D-объектов.
- **Semi-transparency for edited 3D objects** — nanoCAD делает 3D-объекты
  полупрозрачными во время редактирования (Move, 3dMove, Rotate, 3dRotate,
  Scale, Stretch, Mirror), а также при редактировании 3D-объектов с помощью
  grips и при drag-and-drop.
- **Light sources** — в пространстве модели со стилями отображения, поддерживающими
  затенение, модель освещается одним или двумя источниками света позади наблюдателя
  и перемещается вместе с точкой обзора.

## 2. Подходы к моделированию

> **Источник:** [CADtuto — How To Learn 3D Modeling In AutoCAD](https://www.cadtuto.com/learn-3d-modeling/)

Рекомендуемый путь обучения (из источника):

1. Установите 3D-совместимую версию AutoCAD (не AutoCAD LT) и переключите
   рабочее пространство на **3D Modeling**.
2. Изучите навигацию: **3D Orbit, Pan, Zoom, ViewCube**.
3. Освойте базовые 3D-примитивы: **BOX, SPHERE, CYLINDER, EXTRUDE, REVOLVE, SWEEP, LOFT**.
4. Используйте **Boolean operations**: UNION, SUBTRACT, INTERSECT.
5. Редактируйте и улучшайте модели: **FILLET, CHAMFER, SLICE, SHELL, PRESSPULL**.
6. Визуализируйте с помощью **visual styles, materials, basic rendering**.
7. Практикуйтесь на небольших проектах, постепенно увеличивая сложность.

> **Источник:** [CivilTutorials — AutoCAD 3D Modeling](https://civiltutorials.com/autocad-3d-modeling/)

Пошаговый путь для начинающих:

1. Изучите основы 2D-черчения
2. Поймите 3D-интерфейс и инструменты навигации
3. Практикуйтесь с примитивными 3D-объектами: **boxes, spheres, cones**
4. Переходите к **extrusion** и **revolve** для сложных форм
5. Изучите **Boolean operations**: Union, Subtract, Intersect
6. Экспериментируйте с **materials, lighting, rendering**
7. Практикуйтесь на реальных проектах: стул, стол, планировка дома

## 3. Рабочее пространство и виды

> **Источник:** [Autodesk Blog — AutoCAD 2D to 3D Made Easy](https://www.autodesk.com/blogs/autocad/autocad-2d-to-3d-tuesday-tips-with-frank/)

AutoCAD (и nanoCAD как DWG-совместимый аналог) поставляется с двумя встроенными
рабочими пространствами для 3D-потребностей: **3D Basics** и **3D Modeling**.
Инструменты можно найти в любом из них.

- Когда вы сменили рабочее пространство, переключитесь на **3D-вид**.
  Рекомендуется использовать меню **View Controls** для выбора одной
  из предустановленных изометрических видов (например, **SW Isometric**).
- После создания объекта переключите визуальный стиль с **2D Wireframe**
  на что-то более реалистичное, например **Conceptual**.
- Visual Styles: **2D Wireframe, Conceptual, Realistic, Shaded, Shaded with edges,
  Flat shaded, Flat shaded with edges, Wireframe**.

## 4. Пользовательская система координат UCS

> **Источник:** [mycadsite — UCS User Coordinate System in 3D](https://www.mycadsite.com/tutorials/level_3/ucs-3d-dynamic-setting-wcs-autocad-3d-3-12.html)

### Концепция

**WCS (World Coordinate System)** — это система координат по умолчанию,
X — вправо, Y — вверх, Z — к наблюдателю.
**UCS (User Coordinate System)** — это «перенаправление» WCS на основе параметров,
установленных пользователем AutoCAD.

> *Цитата:* «Когда вы работаете в 3D, иногда необходимо изменить плоскость, на которой
> вы рисуете. Например, если вам нужно добавить деталь к стороне стены, вы захотите
> рисовать на этой плоскости. Это как взять лист бумаги с пола (WCS) и приклеить
> его на стену (UCS).»

### Способы установки UCS

1. **3-Point (3 точки)** — самый полезный метод:
   - Точка 1: новое начало координат (X0Y0Z0)
   - Точка 2: направление положительной оси X
   - Точка 3: направление положительной оси Y
   - *Рекомендация источника:* используйте Osnaps для точного выбора точек.

2. **Origin** — перемещает UCS по новой выбранной точке начала координат.
   Плоскость чертежа не сдвигается.

3. **Z-Axis** — выбор двух точек: новое начало и точка для положительной оси Z.

4. **Object (OB)** — выбор 2D-объекта, лежащего на конкретной плоскости чертежа.
   Нужно знать, как объект был нарисован изначально.

5. **View** — UCS автоматически выравнивается с текущим видом, начало остаётся прежним.

6. **X / Y / Z** — поворот UCS вокруг выбранной оси. Указывается точка на оси
   и угол поворота (по правилу правой руки).

7. **Previous** — возвращает к предыдущей настройке UCS. Часто используется в 3D.

8. **Dynamic UCS (F6)** — включение/выключение динамической привязки к граням.
   С включённым Dynamic UCS можно легко рисовать на любой поверхности —
   например, прямоугольник на крыше здания без ручной настройки UCS.

### MUST KNOW из источника

> «3 Point icon — самый лёгкий метод изменения UCS, чтобы он лежал на плоскости.
> Порядок выбора точек важен: 1) X0Y0Z0 (начало), 2) положительная ось X,
> 3) положительная ось Y.»

> «Убедитесь, что включена объектная привязка для точного расположения UCS.»

> «Всегда будьте в курсе, где находится ваш UCS. Убедитесь, что положительный X
> находится там, где вы ожидаете.»

### Применение в 3D-моделировании

> *Цитата:* «UCS особенно мощна в 3D-моделировании, где объекты часто нужно
> манипулировать в трёх измерениях:
> - Выравнивание UCS по 3D-граням: при работе со сложными 3D-моделями выровняйте
>   UCS по конкретным граням тела, используя команду UCS > FACE. Это упрощает
>   рисование, редактирование или измерение элементов на этой грани.
> - Работа в разных плоскостях: UCS позволяет переключаться между разными
>   плоскостями модели (XY, YZ, ZX) путём определения UCS соответствующим образом.
> - Создание UCS для изометрических видов: для изометрических рисунков
>   установите UCS на одну из изометрических плоскостей (Top, Right или Left),
>   используя команду UCS > 3POINT.»

### Просмотр и возврат к WCS

- **UCS icon** — отображает ориентацию текущей UCS в области чертежа.
  Показывает направления осей X, Y, Z и обновляется в реальном времени.
- **PLAN > WORLD** — сбрасывает вид в WCS и выравнивает UCS.
- **UCSMAN** — открывает UCS Manager с графическим интерфейсом для управления
  несколькими настройками UCS (сохранение, восстановление, удаление).

## 5. 2D Sketch Mode — режим эскиза

> **Источник:** [nanoCAD 3D Solid Modeling — 2D Sketch Mode](https://nanocad.com/learning/test-drives/3d/2d-sketch-mode/)

> *Цитата:* «Применяйте **Auto Constraint**, чтобы упростить выбор линий. Можно
> скрыть всю деталь: просто щёлкните правой кнопкой мыши по имени детали в
> 3D History и выберите **Hide**. Если хотите снова сделать её видимой,
> щёлкните правой кнопкой мыши по имени детали в 3D History и выберите **Show**.»

> *Цитата:* «Нажмите **Shift** для ортогонального рисования.»

### Принципы 2D-эскиза для 3D

- **Эскиз рисуется на плоскости** (XY, YZ, XZ или на грани 3D-тела).
- **Контур должен быть замкнутым** (closed profile) — для EXTRUDE, REVOLVE,
  SWEEP, LOFT.
- Используйте **Add Projection** для проецирования существующих рёбер
  на текущую плоскость эскиза — проектируемая грань (ребро) появится
  как линия, которую можно использовать для будущих построений.
- **Auto Constraint** помогает автоматически наложить ограничения (горизонтальность,
  вертикальность, совпадение).
- **Linear** в группе Dimensional — для простановки линейных размеров на эскизе.

### Команда Add Sketch

> *Цитата из источника:* «На вкладке ***3D Tools*** нажмите команду ***Add Sketch***
> (Fig. 10) и нажмите LMB, чтобы выбрать передний торец 3D-модели. Система
> переключилась в режим эскиза. Нарисуйте окружность диаметром ⌀95.»

## 6. 3D Extrude — выдавливание

> **Источник:** [tutorial-autocad.com — EXTRUDE, REVOLVE & Boolean Operations](https://www.tutorial-autocad.com/extrude-revolve-boolean-operations)
> **Источник:** [nanoCAD 3D Extrude Operation](https://nanocad.com/learning/test-drives/3d/3d-extrude-operation/)
> **Источник:** [OpenText BC — Module 18 Solid Modeling](https://opentextbc.ca/autocad3d/chapter/solid-modeling-part-2/)

### Что делает команда

EXTRUDE преобразует **замкнутые LWPOLYLINE, REGION, CIRCLE, ELLIPSE, замкнутые SPLINE**
в 3D-тела путём проецирования в направлении Z на заданное расстояние.

> *Цитата:* «EXTRUDE используется для создания тела путём проецирования замкнутого
> 2D-объекта вдоль оси Z текущего UCS.»

### Синтаксис (командная строка)

```
Command: EXTRUDE↵
Current wire frame density: ISOLINES=4
Select objects: (выбрать 2D-геометрию)
Select objects: ↵
Specify height of extrusion or [Path]: (ввести высоту)
Specify angle of taper for extrusion <0>: (ввести угол сужения)
```

### Параметры

- **Height (высота)** — числовое значение
- **Path (траектория)** — можно использовать путь вместо числовой высоты
  (см. SWEEP)
- **Taper angle (угол сужения)** — для конусного выдавливания
- **Direction (направление)** — можно указать две точки
- **Negative direction** — отрицательное направление (вниз)
- **Both Directions** — выдавливание в обе стороны
- **Cut (Вычитание)** — режим «вычитания» из существующего тела

### Пример из источника nanoCAD (3D Extrude Operation)

> *Цитата:* «1. Нажмите команду ***3D Extrude*** на вкладке ***3D Tools***.
> Подведите курсор к области окружности и подтвердите выбор эскиза левым кликом.
>
> 2. В диалоговом окне ***3D Extrude*** введите значение расстояния 255 и нажмите
> кнопку ***Negative Direction***. Подтвердите с ***OK***.
>
> 3. Затем мы используем опцию вычитания (Cut) в 3D Extrude.
> Переключите ориентацию вида на ***Top NW Isometric*** с помощью ***Locator***.
> Затем переключите визуальный стиль на ***Flat shaded with edges***.
>
> 4. Создадим эскиз на переднем торце объекта. На вкладке ***3D Tools*** нажмите
> команду ***Add Sketch*** и нажмите LMB, чтобы выбрать передний торец 3D-модели.
> Система переключилась в режим эскиза. Нарисуйте окружность диаметром ⌀95.
>
> Снова активируйте операцию ***3D Extrude*** на вкладке ***3D Tools***. Выберите
> область эскиза между большой и малой окружностями кликом. В появившемся окне
> активируйте опцию ***Сut***. В поле Distance введите 255-20 и выберите
> ***Negative control direction***. Нажмите ***OK*** для подтверждения операции.
>
> 5. Аналогично предыдущему шагу, создайте поверхность диаметром ⌀75 и длиной 165.
> 6. Создайте отверстие диаметром ⌀36. Выберите опцию ***Through*** во всплывающем
> меню в параметрах операции 3D Extrude.
> 7. Нарисуйте отверстие диаметром ⌀45 для резьбы M48x2. Введите 45-18.5 в поле Distance.»

### Сравнение EXTRUDE и PRESSPULL

> *Цитата:* «Presspull — мульти-инструмент. Он отличается от Extrude тем, что
> позволяет создавать отверстия и 3D-выдавливания тел после press или pull.»

### Пример EXTRUDE по траектории (Path)

```
Command: EXTRUDE↵
Current wire frame density: ISOLINES=4
Select objects: (выбрать маленькую ОКРУЖНОСТЬ рядом с P1)
Select objects: ↵
Specify height of extrusion or [Path]: P↵
Select extrusion path: (выбрать LWPOLYLINE рядом с P2)
```

### Идеальные варианты использования (из источника learncadforfree)

| Техника | Где применяется |
|---|---|
| EXTRUDE прямоугольника | Стены, балки |
| EXTRUDE окружности | Трубы, колонны |
| REVOLVE кривой | Вазы, чаши, шины |
| Комбинация EXTRUDE + REVOLVE | Сложные составные тела |

### Системная переменная DELOBJ

> *Цитата:* «DELOBJ контролирует, удаляет ли команда EXTRUDE или REVOLVE замкнутый
> объект при выполнении команды. Значение 0 — сохраняет замкнутый многоугольник.»

### Системная переменная ISOLINES

> *Цитата:* «Установите системную переменную ISOLINES в 48, чтобы получить более
> гладкое отображение искривлённых поверхностей.»

## 7. 3D Revolve — вращение

> **Источник:** [tutorial-autocad.com — EXTRUDE, REVOLVE & Boolean Operations](https://www.tutorial-autocad.com/extrude-revolve-boolean-operations)
> **Источник:** [nanoCAD — The 3D Revolve Operation](https://nanocad.com/learning/test-drives/3d/the-3d-revolve-operation/)

### Что делает команда

> *Цитата:* «Команда Revolve похожа на использование Extrude с радиальной
> траекторией (но вам не требуется объект-траектория при использовании Revolve).
> Можно вращать замкнутые LWPOLYLINE, REGION, CIRCLE, ELLIPSE и замкнутые SPLINE
> в 3D-тела. Эта команда практична для моделирования объектов, которые должны
> быть изготовлены на токарном станке (например, валы).»

### Способы задания оси вращения

1. **Две точки** (по умолчанию) — укажите две точки, чтобы задать ось вращения.
2. **Object** — выберите LINE на нужной оси и используйте опцию Object
   команды Revolve, чтобы указать ось.

### Параметры

- **Angle of revolution** — угол от 1° до 360°
- **Axis of revolution** — две точки или существующий объект (LINE)
- **Direction of revolution** — CW (по часовой стрелке) или CCW (против)
- **TgSolid (Tangent to Solid)** — выберите существующее тело, к которому
  будет присоединён результат (nanoCAD-специфика)

### Пример из источника nanoCAD (3D Revolve Operation)

> *Цитата:* «1. Создайте эскиз на плоскости YZ.

## 8. 3D Sweep — сдвиг по траектории

> **Источник:** [Autodesk — About Creating a Solid or Surface by Sweeping](https://help.autodesk.com/cloudhelp/2023/ENU/AutoCAD-Core/files/GUID-89EE26BD-D032-430C-AF3C-DF2BAD657391.htm)
> **Источник:** [learnvern — How to Use Sweep and Loft Command](https://www.learnvern.com/autocad-tutorial/sweep-and-loft-command)

### Что делает команда

> *Цитата:* «Команда SWEEP создаёт тело или поверхность путём прохождения формы
> профиля (объекта, который сдвигается) по указанной траектории. Когда вы сдвигаете
> профиль по траектории, профиль перемещается и выравнивается **normal (perpendicular)**
> к траектории. **Открытые профили создают поверхности**, а **замкнутые кривые создают тела
> или поверхности**.»

### Требования

- Профиль и траектория должны существовать в модели.
- **Можно сдвигать более одного объекта-профиля** по траектории.
- Открытый профиль → поверхность; замкнутый профиль → тело или поверхность.

### Параметры

- **Alignment (Выравнивание)** — если профиль не в той же плоскости, что и траектория
  сдвига, укажите, как профиль выравнивается с траекторией.
- **Mode** — устанавливает, создаёт ли сдвиг поверхность или тело.
- **Base Point** — укажите базовую точку на профиле, чтобы сдвигать по профилю.
- **Twist angle (угол закручивания)** — для винтовых/спиральных форм.

### Идеальное применение

> *Цитата:* «Команда sweep особенно полезна при создании форм с острыми углами
> и рёбрами, таких как окружности и эллипсы, потому что позволяет изменять
> радиус кривизны в любой точке вдоль траектории.»

## 9. 3D Loft — лофтинг по сечениям

> **Источник:** [Autodesk — About Creating a Solid or Surface by Lofting](https://help.autodesk.com/cloudhelp/2024/ENU/AutoCAD-MAC-Core/files/GUID-C6225BFB-B2AA-4603-9013-E9C2FB640653.htm)
> **Источник:** [Autodesk — LOFT (Command)](https://help.autodesk.com/cloudhelp/2024/ENU/AutoCAD-MAC-Core/files/GUID-0A041818-2E32-4212-A3D8-CE0361C3D229.htm)

### Что делает команда

> *Цитата:* «LOFT создаёт 3D-тело или поверхность в пространстве между несколькими
> поперечными сечениями. Поперечные сечения определяют форму результирующего тела.
> Нужно указать как минимум **два** поперечных сечения.»

### Параметры сечений

- **Cross-section profiles** — серия поперечных сечений, определяющих форму
  нового 3D-объекта.
- **Cross-section profiles могут быть открытыми или замкнутыми**:
  открытые создают поверхности, замкнутые — тела или поверхности.
- **Cross-section profiles могут быть планарными или непланарными**, а также
  edge-подобъектами.
- Режим (mode) — выбрать, создавать ли поверхность или тело.

### Способы управления формой

1. **Cross Sections Only** — создание лофт-объектов без использования направляющих
   или траекторий.

2. **Path (Траектория)** — указание траектории для лофт-операции для большего
   контроля над формой лофт-объекта.
   > «Для лучших результатов начните кривую траектории на плоскости первого
   > поперечного сечения и закончите её на плоскости последнего поперечного сечения.»
   > «Кривая траектории должна пересекать все плоскости поперечных сечений.»

3. **Guide curves (Направляющие кривые)** — указание направляющих кривых
   для сопоставления точек на соответствующих поперечных сечениях.
   > «Этот метод предотвращает нежелательные результаты, такие как складки
   > на результирующем 3D-объекте.»
   > «Каждая направляющая кривая должна:
   >   - пересекать каждое поперечное сечение,
   >   - начинаться на первом поперечном сечении,
   >   - заканчиваться на последнем поперечном сечении.»

### Настройки (Settings)

> «При создании лофт-объекта можно регулировать его форму, указав, как профиль
> проходит через поперечные сечения (например, **острая** или **плавная** кривая).
> Вы также можете изменить настройки позже в палитре Properties.»

### Применение (из источника)

> «LOFT может использоваться для создания многих различных форм, таких как сфера,
> эллипс или даже треугольник. Также может использоваться для создания изогнутых
> поверхностей, как у автомобилей и самолётов.»

> «Процесс лофтинга — это метод 3D-моделирования, который берёт две или более
> поверхности и создаёт новую поверхность, соединяющую их. Лофт-поверхность
> создаётся путём пересечения двух поверхностей по их общему ребру или
> граничным кривым, создавая единую поверхность с органичным видом.»

### Удаление исходных кривых

> «Чтобы автоматически удалить поперечные сечения, направляющие и траектории,
> используйте DELOBJ. Если ассоциативность поверхности включена, DELOBJ
> игнорируется, и исходная геометрия не удаляется.»

## 10. Булевы операции

> **Источник:** [tutorial-autocad.com — EXTRUDE, REVOLVE & Boolean Operations](https://www.tutorial-autocad.com/extrude-revolve-boolean-operations)
> **Источник:** [OpenText BC — Module 18 Solid Modeling](https://opentextbc.ca/autocad3d/chapter/solid-modeling-part-2/)
> **Источник:** [Autodesk — About Creating Basic 3D Solids and Walls](https://help.autodesk.com/cloudhelp/2021/ENU/AutoCAD-Core/files/GUID-A6C31606-37F1-4D83-A325-C1399488AD42.htm)

### Что это

> *Цитата:* «Boolean-операции обычно используются после создания основных
> элементов модели с помощью Extrude или Revolve над 2D-геометрией.»

### Три операции

1. **UNION (Объединение)** — создаёт новое ТЕЛО, которое включает в себя
   объединённые объёмы выбранных ТЕЛ (как сварка деталей вместе).
2. **SUBTRACT (Вычитание)** — удаляет перекрывающиеся объёмы выбранных ТЕЛ
   из других ТЕЛ (как фрезерование нежелательного материала).
3. **INTERSECT (Пересечение)** — генерирует новое ТЕЛО, которое включает
   только общий объём выбранных ТЕЛ.

### Синтаксис SUBTRACT (двухэтапная команда)

```
Pick Modify + Solids Editing + Subtract.
Select the SOLID near P1 as the object to subtract from and press.
Select the SOLID(s) to subtract and press.
```

> *Цитата:* «SUBTRACT — это двухэтапная команда, в которой сначала предлагается
> выбрать ТЕЛА, из которых вы будете вычитать. Если вы выберете более одного
> ТЕЛА для вычитания, эти объекты сначала объединяются вместе (как если бы вы
> использовали Union). Затем вам предлагается выбрать ТЕЛА для вычитания,
> и вы можете выбрать более одного ТЕЛА в одной операции.»

### Практика из источника (tutorial-autocad.com)

> *Цитата:* «15. Pick Modify + Solids Editing + Subtract. Select the large outer
> SOLID near P1 as the object to subtract from then press to continue. Select the
> smaller inner SOLID near P2 as the object to subtract and press.
> ...
> 26. Pick Modify + Solids Editing + Subtract. Select the large outer SOLID near P1
> as the object to subtract from then press to continue. Select the smaller inner
> SOLID near P2 as the object to subtract and press.
> ...
> 37. Pick Modify + Solids Editing + Union. Select all SOLIDs visible on-screen and press.
> ...
> 57. Pick Modify + Solids Editing + Intersect. Type ALL to select all three new
> SOLID objects and press to complete the task.
>
> Вы создали эту полную модель путём Intersect экструдированных 2D-профилей
> (силуэтов) каждого ортогонального вида в 2D-чертеже.»

### Альтернативный путь: REGION + Boolean

> «Можно создать REGION из 2D-объектов, затем выполнить SUBTRACT над REGIONs,
> чтобы создать новый REGION (с дырками), а затем EXTRUDE его в тело.»

### Принципы (из OpenText BC)

> «Экструзия включает рисование замкнутого объекта и, используя команду EXTRUDE
> или PRESSPULL, проецирование его в направлении Z на заданное расстояние.
> Замкнутый объект может быть 2D-многоугольником, окружностью или эллипсом.
> Экструдированные тела затем могут быть объединены командой UNION или вычтены
> командой SUBTRACT для формирования финальной модели тела.»

> «Не все модели тел могут быть экструдированы. Модель тела, которая симметрична,
> может быть создана путём вращения замкнутого объекта.»

## 11. Fillet и Chamfer на рёбрах 3D тел

### FILLET EDGE (Скругление рёбер)

> *Цитата:* «Команда Fillet в 3D используется для создания гладких рёбер.»

> 1. Введите FILLETEDGE.
> 2. Продолжайте выбирать рёбра.
> 3. Нажмите Enter для завершения.»

#### Loop (Петля)

> «Опция Loop выбирает все рёбра вокруг базовой поверхности.

> После выбора ребра петли вам предлагается принять текущий выбор

> *Цитата:* «Команда Chamfer в AutoCAD 3D используется для создания скошенных рёбер.»

#### Пошаговый алгоритм

1. Введите **CHAMFEREDGE** в командной строке.
2. Введите **D** или **Distance**.
3. Нажмите Enter.
4. Укажите **Distance1** (например, 2).
5. Нажмите Enter.
6. Укажите **Distance2** (например, 2).
7. Нажмите Enter.

9. Нажмите Enter.

#### Loop Chamfer

> «Чтобы продолжить применение CHAMFER к другим рёбрам с теми же Distance1

### Base Surface (Базовая поверхность) при CHAMFER

> *Цитата:* «Chamfer выдаёт другие подсказки при выборе ТЕЛ по сравнению с выбором

>
> После того, как вы установите, какая поверхность является базовой, вы можете выбрать
> несколько отдельных рёбер для снятия фаски относительно этой поверхности.
> Опция Loop выбирает все рёбра вокруг базовой поверхности.»

### Разница между FILLET и CHAMFER (из CanadaCAD)

> *Цитата:* «Скругление (fillet) представляет собой изогнутое или закруглённое ребро
> между двумя поверхностями, тогда как фаска (chamfer) представляет прямое скошенное
> ребро. Понимание этих различий важно для достижения желаемых результатов
> моделирования.»

> «Скругления необходимы в 3D-моделировании, особенно в механическом дизайне.
> Скругление обычно представляет плавный, закруглённый переход между двумя рёбрами,
> помогая как в эстетической привлекательности, так и в структурной целостности.
> Используя скругления, дизайнеры могут минимизировать концентрации напряжений,
> тем самым повышая долговечность механических деталей.»

> *Источник:* [Autodesk — To Modify a Fillet or Chamfer on a 3D Solid](http://help.autodesk.com/cloudhelp/2021/ENU/AutoCAD-Core/files/GUID-0C36AAC5-EFC2-4B79-A053-8B4C3C6D5D1E.htm)

## 12. 3D Array — массивы в 3D

> **Источник:** [tutorial-autocad.com — 3D Operations](https://www.tutorial-autocad.com/lesson-3-3d-operations)
> **Источник:** [eng.libretexts.org — Module 9: Arraying and Mirroring 3D Models](https://eng.libretexts.org/Bookshelves/Introductory_Engineering/CAD_Skills_for_First-Year_Engineers%3A_A_Hands-On_Guide_to_Sketching_Drafting_and_Prototyping/03%3A_Computer_Aided_Design_(CAD)/3.03%3A_AutoCad_resources/3.3.02%3A_Part_2/3.3.2.03%3A_Module_9-_Arraying_and_Mirroring_3D_Models)
> **Источник:** [CADtuto — 3DARRAY Command In AutoCAD](https://www.cadtuto.com/3darray/)
> **Источник:** [nanoCAD — The 3D Array](https://nanocad.com/learning/test-drives/3d/the-3d-array/)

### Зачем нужен 3D Array

> *Цитата:* «При конструировании 3D-моделей единственный раз, когда можно использовать
> 2D-команды ARRAY и MIRROR, — это когда все выбранные объекты лежат на 2D-плоскости
> текущего UCS. Для массирования или зеркалирования объектов в 3D-пространстве
> должны использоваться соответствующие 3D-команды.»

### 3DARRAY — типы массивов

> *Цитата:* «3DARRAY создаёт **неассоциативные 3D-массивы** — либо **прямоугольные (X, Y, Z)**,
> либо **полярные (вокруг оси/центра)** — путём создания копий выбранных объектов
> и размещения их в 3D-сетке или круговом шаблоне.
> «Неассоциативный» означает, что копии независимы от оригинала: редактирование
> исходного объекта не обновляет копии.»

#### Прямоугольный 3D Array (сетка: колонки × строки × уровни)

1. Выберите объекты, которые хотите разместить в массиве (тела, поверхности, блоки и т.д.).
2. Запустите команду: введите `3darray` и нажмите Enter.
3. Выберите тип: **Rectangular**.
4. Укажите количество элементов в каждом направлении:
   - Колонки (X)
   - Строки (Y)
   - Уровни (Z) — *третье измерение 3D-массива называется **level***
5. Укажите расстояние или общий extent для каждого направления.

Пример: 4 колонки × 3 строки × 2 уровня = 4 × 3 × 2.

#### Полярный 3D Array (круговой вокруг центра, с уровнями)

1. Запустите `3darray` и выберите **Polar**.
2. Укажите центральную точку вращения.
3. Введите количество элементов по кругу.
4. Укажите угол заполнения (360° для полного круга, меньше — для дуг).
5. Для 3D-полярных массивов может быть предложено ввести количество уровней
   (направление Z) и расстояние между уровнями.

#### Синтаксис 3DARRAY

```
Command: 3DARRAY↵
Initializing… 3DARRAY loaded.
Select objects: (выбрать объекты)
Select objects: ↵
Enter the type of array [Rectangular/Polar]: P↵
Enter the number of items in the array: 8↵
Specify the angle to fill (+=ccw, -=cw) <360>: ↵
Rotate arrayed objects? [Yes/No]: ↵
Specify center point of array: (указать точку)
Specify second point on axis of rotation: (указать вторую точку оси)
```

### Отличия от современной команды ARRAY

> «3darray: создаёт неассоциативные копии. Вы не можете редактировать массив

### Альтернативные методы (из источника)

- Используйте современную команду ARRAY для ассоциативных массивов.
- Используйте **COPY** с опцией Multiple для размещения повторений, где
  требуется точное расстояние или нерегулярное размещение.
- Используйте вставку **BLOCK**, а затем выполните массив блока.
- Для сложных траекторий используйте **ARRAYPATH**.
- Используйте скрипты или dynamic blocks для параметризованных модульных повторов.

### Советы по эффективному использованию (из CADtuto)

- Преобразуйте повторяющиеся части в блоки перед массированием, чтобы уменьшить
  размер файла и улучшить производительность.
- Используйте preview перед подтверждением, чтобы убедиться в правильности
  ориентации и расстояния.
- Если вам нужны дальнейшие изменения, предпочитайте ассоциативный ARRAY.
- Следите за UCS и работайте в системе координат, которая упрощает указание
  расстояний.
- Для физического изготовления или BIM-процессов убедитесь, что единицы
  и расстояния соответствуют требованиям модели.

## 18. Ассоциативность моделей и чертежей

> **Источник:** [nanoCAD — 3D Solid Modeling Features](https://nanocad.com/products/nanocad-3d-modeling-module/features/)

> «Ассоциативная связь моделей и их проекций позволяет пользователям сосредоточиться
> на проектировании в 3D-среде, уделяя внимание 2D-графике в основном для оформления

- Можно моделировать в 3D и не беспокоиться о ручном обновлении 2D-видов

  изменении 3D-модели
- Можно добавлять аннотации (размеры, шероховатости, базы) в 2D-графике,
  и они останутся связанными с 3D-моделью

## 20. Типовые ошибки и их устранение

> **Источник:** [Autodesk — Sketch is not recognized as closed profile](https://www.autodesk.com/support/technical/article/caas/sfdcarticles/sfdcarticles/The-sketch-in-Fusion-360-is-not-recognized-as-closed-profile.html)
> **Источник:** [CADtuto — How To Learn 3D Modeling In AutoCAD](https://www.cadtuto.com/learn-3d-modeling/)
> **Источник:** [Mechnexus — 10 CAD Modeling Mistakes](https://mechnexus.com/10-cad-modeling-mistakes-youre-probably-making/)
> **Источник:** [SelfCAD — 11 Common 3D Modeling Mistakes](https://www.selfcad.com/blog/common-3d-modeling-mistakes-and-how-to-avoid-them)

### Ошибка: EXTRUDE не выполняется или создаёт пустоту

> *Цитата:* «Проблема: EXTRUDE fails or creates nothing. Решение: убедитесь, что
> профиль замкнут без зазоров. Используйте BOUNDARY или REGION, чтобы создать
> замкнутую область. Запустите OVERKILL для очистки перекрывающихся линий.»

### Ошибка: Boolean-операции дают неожиданные результаты

> *Цитата:* «Проверьте нормали и убедитесь, что тела правильно пересекаются;
> используйте INTERSECT, чтобы увидеть перекрытие. Очистите геометрию
> и используйте SOLIDEDIT > Separate при необходимости.»

### Ошибка: профиль не распознаётся как замкнутый

> *Цитата:* «Открытые профили возникают из-за крошечных зазоров, перекрывающихся
> линий или несовпадающих конечных точек. Включите Sketch → Show Profile, увеличьте
> масштаб и соедините конечные точки инструментами Coincident, Extend или Trim.»

#### Решения для закрытия профиля

1. **Проверьте, что нет открытых точек.** Открытые точки могут быть обозначены
   белой точкой. Измените эскиз, чтобы убедиться, что профиль замкнут, добавив
   constraints или геометрию.
2. **Если эскиз подтверждён замкнутым, но проблема остаётся:**
   - Скопируйте геометрию и вставьте в новый эскиз.
   - Некоторые relations эскиза могут быть потеряны, но теперь он должен
     показывать замкнутые профили.
3. **Убедитесь, что эскиз на одной плоскости.** Если эскиз не на одной плоскости,
   спроецируйте эскиз на одну плоскость.
4. **Используйте сторонние аддоны** (SketchChecker, Fill Gaps in Sketch).

### Ошибка: открытый профиль

> *Цитата:* «Запустите PEDIT или JOIN, чтобы сварить сегменты, или используйте
> BOUNDARY, чтобы создать замкнутую область. Увеличьте масштаб и используйте
> OVERKILL для удаления дублирующихся сегментов.»

### Ошибка: 3DARRAY даёт неправильную ориентацию

> «Проверьте UCS и selection base point. Массив использует текущую ориентацию
> координат — установите UCS так, чтобы X/Y/Z выравнивались с предполагаемыми
> направлениями.»

### Ошибка: 3DARRAY недоступен

> «Убедитесь, что ваша версия поддерживает 3darray и что 3D-наборы инструментов
> включены. Попробуйте современную команду ARRAY, если 3darray отсутствует.»

### Топ-11 типовых 3D-ошибок (SelfCAD)

1. **Not Using a Reference Point** — отсутствие опорной точки. Используйте
   center point или origin, чтобы добавлять другие точки и держать модель
   организованной.
2. **Misaligned Geometry** — несоосная геометрия. Это может вызвать проблемы
   при текстурировании, освещении, а также привести к неправильным результатам
   Boolean-операций.
3. **Duplicate Vertices/Edges** — дублирование вершин или рёбер может вызвать
   проблемы с UV mapping, smoothing и анимацией.
4. **Poor Topology** — плохая топология. Проявляется в виде Ngon (полигонов
   с более чем 4 сторонами), double-sided faces, loose vertices.
5. **Wrong Scale** — неправильный масштаб. Модель может выглядеть правильно
   на экране, но быть неправильной при печати или в реальной жизни.
6. **Incorrect Use of Symmetry and Mirroring** — некорректное использование
   симметрии и зеркалирования приводит к искажённым пропорциям.
7. **Overusing Modifiers** — чрезмерное использование модификаторов (цвета,
   текстуры, освещение) делает модель «постановочной» или «фейковой».
8. **Not Naming Objects Properly** — неправильное именование объектов вызывает
   проблемы при анимации, рендере, экспорте.
9. **Using the Wrong Units** — использование неправильных единиц измерения
   (метрические, имперские, корпусные).
10. **Opening a Lot of Projects at The Same Time** — 3D-моделирование очень
    ресурсоёмко; одновременное открытие множества проектов замедляет работу.
11. **Creating Textures That are Not Optimized for 3D Printing** — неоптимизированные
    текстуры для 3D-печати приводят к нерегулярной поверхности или плохому
    качеству печати.

### Топ-10 CAD-ошибок (Mechnexus)

| Категория | Частые примеры | Как избежать |
|---|---|---|
| Основа | Не заданы единицы; плохое использование шаблонов | Используйте стандартизированные шаблоны с предопределёнными единицами, слоями, стилями |
| Точность | «На глаз» вместо привязок; зазоры в геометрии | Всегда используйте OSNAP и polar tracking; используйте OVERKILL и PEDIT |

| Аннотации | Ручное редактирование/взрыв размеров; непонятные примечания | Используйте ассоциативные размеры; предопределите и используйте согласованные стили текста и размеров |

### Основные принципы (из источников)

1. **Plan Before You Model** — потратьте несколько минут на настройку единиц,
   шаблонов и слоёв в начале, это сэкономит часы переработки впоследствии.
2. **Embrace "Associative" Everything** — используйте ассоциативные размеры
   и блоки. Это сохраняет модель «умной»: когда геометрия меняется, размеры
   и экземпляры блоков обновляются автоматически, предотвращая несоответствия.

### Из CADtuto

- Запомните ключевые команды: **EXTRUDE, REVOLVE, UNION, SUBTRACT, FILLET**.
- Используйте горячие клавиши и пользовательские алиасы для ускорения
  (редактируйте файл acad.pgp).
- Начинайте с чистых 2D-эскизов — хорошие эскизы делают 3D-моделирование
  быстрым и предсказуемым.
- Сохраняйте итерации часто и используйте «save as» для создания бэкапов.
- Используйте референсные изображения или размерные чертежи для поддержания точности.
- Научитесь использовать Viewports в layouts для представления нескольких видов
  в разных масштабах.
- Практикуйтесь в моделировании реальных объектов для развития пространственного
  мышления.
- Научитесь правильно экспортировать в STL для 3D-печати (проверьте нормали
  и watertightness).

### Из imaginxp

- **Over-detailing** — избегайте избыточных fillets/chamfers, увеличивающих
  размер файла. Используйте feature suppression для некритичных деталей.
- **Dimension constraints** — добавляйте полностью определённые constraints
  (horizontal/vertical, equal) в режиме Sketch, фиксируя 5-7 параметров
  на эскиз.
- **Relative coordinates** — используйте endpoint-to-endpoint привязки,
  а не абсолютные координаты, для более быстрого размещения.
- **Update tool** — используйте Update после редактирования эскизов, чтобы
  перестроить дерево и распространить изменения.
- **Material library validation** — выбирайте точные свойства материалов
  из библиотеки для точности симуляций.
- **Fillet radii** — применяйте радиусы 1-2 мм на рёбрах, чтобы избежать
  концентрации напряжений и улучшить качество 3D-печати.
- **Instance referencing** — используйте instance referencing для идентичных
  деталей вместо дублирования тел, чтобы уменьшить размер файла.
- **Tolerance stack-up** — используйте калькуляторы stack-up для проверки
  зазоров ±0.1 мм и обеспечения 95% точности посадки.

### Соответствие циклов работы MCP операциям nanoCAD

#### Цикл 1: Создание тела вращения (REVOLVE)

```python
# 1. Создать 2D профиль (полилиния, окружность и т.д.)
# Пример: контур вала — полу-окружность диаметром 50
handle_profile = create_polyline(vertices=[
    [0, 0], [25, 0], [25, 100], [0, 100]
], closed=True, layer="Профиль_Вал")

# 2. REVOLVE вокруг оси X (axis_x=1, axis_y=0, axis_z=0)
# Угол 360° (полный оборот)
handle_body = revolve_solid(
    handle=handle_profile,
    axis_x=1.0, axis_y=0.0, axis_z=0.0,  # ось вращения
    dir_x=0.0, dir_y=0.0, dir_z=1.0,       # направление «вверх» для профиля
    angle=360.0
)
```

#### Цикл 2: Extrude с Cut (создание паза/отверстия)

```python
# 1. Создать основное 3D тело
handle_box = create_box(x=200, y=100, z=50)  # 200×100×50 мм

# 2. Создать 2D эскиз отверстия
handle_circle = create_circle(cx=100, cy=50, radius=20, layer="Отверстие")

# 3. Extrude отверстия
handle_hole = extrude_solid(handle=handle_circle, height=60)  # проходит насквозь

# 4. SUBTRACT (вычитание отверстия из тела)
handle_with_hole = boolean_subtract(handle1=handle_box, handle2=handle_hole)
```

#### Цикл 3: 3D Polar Array (массив отверстий по кругу)

```python
# 1. Создать одно отверстие как 3D тело
handle_hole = create_cylinder(radius=5, height=20)
move_solid(handle=handle_hole, dx=100, dy=0, dz=0)  # сдвинуть на радиус

# 2. Создать массив из 6 отверстий через 60° по кругу
# ВАЖНО: array_3d работает в декартовых координатах, для полярного массива
# вокруг произвольной оси используйте другие подходы
# (комбинация rotate + копирование, или execute_command с 3DARRAY)
```

#### Цикл 4: FILLET EDGE на 3D теле

```python
# 1. Создать тело с острыми рёбрами
handle_box = create_box(x=100, y=100, z=50)

# 2. Скруглить все рёбра радиусом 5 мм
handle_fillet = fillet_edge(handle=handle_box, radius=5.0)
```

#### Цикл 5: Sweep (сдвиг профиля по траектории)

```python
# 1. Создать профиль (окружность)
handle_profile = create_circle(cx=0, cy=0, radius=5)

# 2. Создать траекторию (например, спираль или ломаную)
handle_path = create_polyline(vertices=[
    [0, 0, 0], [50, 0, 50], [100, 0, 0]
])

# 3. Sweep
handle_swept = sweep_solid(
    profile_handle=handle_profile,
    path_handle=handle_path
)
```

#### Цикл 6: 3D Mirror (зеркалирование через плоскость)

```python
# 1. Создать тело
handle_box = create_box(x=100, y=50, z=30)

# 2. Зеркалировать через плоскость XY (плоскость Z=0)
handle_mirrored = mirror_3d(
    handle=handle_box,
    p1_x=0, p1_y=0, p1_z=0,
    p2_x=1, p2_y=0, p2_z=0,
    p3_x=0, p3_y=1, p3_z=0
)
```

### Важные замечания по работе через MCP

1. **3D операции (`revolve_solid`, `extrude_solid`, `boolean_*`, `fillet_edge`,
   `chamfer_edge`, `sweep_solid`, `loft_solid`, `array_3d`, `mirror_3d`,
   `fillet_edge`, `chamfer_edge`) требуют .NET engine** — должны работать
   в nanoCAD с загруженным плагином CadEngine.Plugin.

2. **Системная переменная DELOBJ** (удалять ли исходный 2D профиль после
   extrude/revolve) — поведение зависит от реализации в .NET engine.
   Сохраняйте handle профиля на случай, если нужно повторить операцию.

3. **Системная переменная ISOLINES** — плотность каркаса на искривлённых
   поверхностях. По умолчанию 4, рекомендуется 48 для гладкого отображения.

4. **3DARRAY в MCP представлен как `array_3d`** и работает только с
   прямоугольными массивами. Для полярных массивов вокруг произвольной оси
   используйте `execute_command` с командой `3DARRAY` или комбинацию
   `copy_entity` + `rotate_entity`.

5. **UCS** — в MCP нет прямого инструмента для управления UCS. Используйте
   `execute_command` с командой `UCS`, или работайте в WCS, или создавайте
   2D эскизы в WCS, а затем размещайте их в 3D через `move_solid` /
   `move_entity`.

6. **Visual Styles** — `set_3d_view(direction, render_mode)` позволяет
   задать направление и режим отображения. Например:
   - `set_3d_view(direction="SW_ISOMETRIC", render_mode="CONCEPTUAL")`
   - `set_3d_view(direction="FRONT", render_mode="REALISTIC")`
