## 1. ОСНОВНЫЕ ПОНЯТИЯ

### Терминология
- **Номинальный размер (D)** — размер, относительно которого определяются предельные размеры
- **Действительный размер** — размер, установленный измерением
- **Предельные размеры** — наибольший D_max и наименьший D_min допустимые размеры
- **Верхнее отклонение**: отверстие ES = D_max - D; вал es = d_max - d
- **Нижнее отклонение**: отверстие EI = D_min - D; вал ei = d_min - d
- **Допуск (T)**: T_D = ES - EI (отверстие); T_d = es - ei (вал)
- **Посадка** — характер соединения, определяемый разностью размеров до сборки
- **Зазор (S)**: S = D - d; S_max = D_max - d_min; S_min = D_min - d_max
- **Натяг (N)**: N = d - D; N_max = d_max - D_min; N_min = d_min - D_max

### Единица допуска (i)
- Для размеров от 1 до 500 мм: i = 0.45·∛D + 0.001·D, мкм
- Для размеров от 500 до 3150 мм: I = 0.004·D + 2.1, мкм
- D — среднее геометрическое границ интервала: D = √(D₁·D₂)

## 6. РАСЧЁТ РАЗМЕРНЫХ ЦЕПЕЙ

### Метод полной взаимозаменяемости (max/min)
- A_Δ = ΣA_i_ув - ΣA_i_ум (замыкающее звено)
- T_AΔ = ΣT_Ai (допуск замыкающего звена)
- Δ₀AΔ = ΣΔ₀A_i_ув - ΣΔ₀A_i_ум (координата середины)
- Верхнее откл. ES_AΔ = Δ₀AΔ + T_AΔ/2
- Нижнее откл. EI_AΔ = Δ₀AΔ - T_AΔ/2

### Метод неполной взаимозаменяемости (теоретико-вероятн.)
- T_AΔ = t·√(Σ(λ_i·T_Ai)²), где t = 3 (p=0.9973), λ_i = 1/3 для нормального закона
- Для 10-15 звеньев даёт расширение допусков в 1.5-2 раза

### Регулирование и пригонка
- Используется при большом числе звеньев (>6) и узком поле T_AΔ
- Размер компенсатора: A_к = A_Δ_расч - A_Δ_треб
- Величина компенсации: δ_к = T_AΔ_расч - T_AΔ_треб
