# nanoCAD 26 - Слои и свойства объектов

## Управление слоями

### Создание слоя
- **Команда:** LAYER или LA
- **Параметры:**
  - Имя
  - Цвет (ACI, True Color, Color Book)
  - Тип линии
  - Толщина линии
  - Прозрачность
  - Состояние (вкл/выкл, заморожен, заблокирован)

### Свойства слоя

| Свойство | Описание | Значения |
|----------|----------|----------|
| On/Off | Видимость | Да/Нет |
| Freeze/Thaw | Заморозка | Да/Нет |
| Lock/Unlock | Блокировка | Да/Нет |
| Plot | Печать | Вкл/Выкл |
| Color | Цвет | ACI 1-255, True Color |
| Linetype | Тип линии | Continuous, Dashed, Center... |
| Lineweight | Толщина | 0.00 - 2.11 мм |
| Transparency | Прозрачность | 0-90 |

### Команды управления слоями

| Команда | Описание |
|---------|----------|
| LAYER | Диалог управления слоями |
| LAYERSTATE | Сохранение/восстановление состояний |
| LAYMCH | Изменение слоя объекта |
| LAYISO | Изоляция слоя |
| LAYUNISO | Отмена изоляции |
| LAYOFF | Отключение слоя |
| LAYON | Включение всех слоёв |
| LAYFRZ | Заморозка слоя |
| LAYTHW | Разморозка всех слоёв |
| LAYDEL | Удаление слоя |
| LAYMRG | Объединение слоёв |
| LAYCUR | Сделать слой текущим |

### Стандартные слои (ГОСТ)

| Имя слоя | Назначение |
|----------|-----------|
| 0 | Базовый слой |
| DEFPOINTS | Точки определения (не печатается) |
| Стены | Стены здания |
| Окна | Окна и двери |
| Размеры | Размерные линии |
| Текст | Текстовые надписи |
| Штриховка | Штриховки |
| Оси | Оси конструкций |

## Свойства объектов

### Панель свойств
- **Команда:** PROPERTIES или PR
- **Отображение:** Выбранный объект → Все свойства

### Общие свойства
| Свойство | Описание |
|----------|----------|
| Color | Цвет |
| Layer | Слой |
| Linetype | Тип линии |
| Linetype scale | Масштаб типа линии |
| Lineweight | Толщина |
| Transparency | Прозрачность |
| Hyperlink | Гиперссылка |
| Thickness | Толщина (2D экструзия) |

### Свойства геометрии
| Тип объекта | Свойства |
|-------------|----------|
| Line | Start Point, End Point, Length, Angle |
| Circle | Center, Radius, Diameter, Circumference, Area |
| Arc | Center, Radius, Start Angle, End Angle, Area |
| Polyline | Vertices, Closed, Length, Area |
| Spline | Fit Points, Control Points, Degree |

### Быстрые свойства
- **Команда:** QUICKPROPERTIES
- Отображение свойств при выборе
- Настройка отображаемых свойств

## Стили линий

### Типы линий
| Тип | Описание | ГОСТ |
|-----|----------|------|
| Continuous | Сплошная | Сплошная |
| Dashed | Штриховая | Штриховая |
| Hidden | Невидимая | Пунктирная |
| Center | Центровая | Штрих-пунктирная |
| Phantom | Фантомная | Двойная штрих-пунктирная |
| Dashdot | Штрих-пунктирная | Штрих-пунктирная |
| Border | Граница | Сплошная |

### Масштаб типов линий
- Глобальный: LTSCALE
- Объектный: Linetype scale в свойствах

## Цвета

### Палитра цветов ACI
- 256 стандартных цветов
- 1-7: 기본ные (красный, жёлтый, зелёный, голубой, синий, фиолетовый, белый)
- 8-255: расширенные

### True Color
- RGB (0-255, 0-255, 0-255)
- HSL (Hue, Saturation, Luminance)

### Цветовые книги
- RAL (порошковые покрытия)
- Pantone (полиграфия)
- DIC (промышленность)

## Прозрачность

### Настройка
- Общая: TRANSPARENCY
- Объектная: в свойствах
- Слоёвая: в настройках слоя

### Значения
- 0: Непрозрачный
- 90: Почти прозрачный
