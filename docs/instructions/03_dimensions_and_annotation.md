# nanoCAD 26 - Размеры и аннотации

## Типы размеров

### Линейные размеры
| Команда | Описание |
|---------|----------|
| DIMLINEAR | Горизонтальный/вертикальный размер |
| DIMALIGNED | Выровненный размер (вдоль линии) |
| DIMBASELINE | базовый размер (от общей точки) |
| DIMCONTINUE | Цепной размер |

### Угловые размеры
| Команда | Описание |
|---------|----------|
| DIMANGULAR | Угол между двумя линиями |

### Радиальные размеры
| Команда | Описание |
|---------|----------|
| DIMRADIUS | Радиус окружности/дуги |
| DIMDIAMETER | Диаметр окружности |
| DIMCENTER | Центральные линии |

### Ординатные размеры
| Команда | Описание |
|---------|----------|
| DIMORDINATE | Координатная точка |

### Листовые размеры
| Команда | Описание |
|---------|----------|
| LEADER | Выноска с текстом |
| MLEADER | Многострочная выноска |
| TOLERANCE | Допуск |
| GD&T | Геометрические допуски |

## Стили размеров

### Основные параметры стиля
- **Линии:** цвет, тип, толщина, отступы
- **Стрелки:** тип, размер
- **Текст:** стиль, высота, расположение
- **Единицы:** формат, точность
- **Допуски:** формат, точность

### Создание стиля
1. Команда DIMSTYLE (D)
2. Создать новый стиль
3. Настроить параметры вкладок:
   - Lines (Линии)
   - Symbols and Arrows (Символы и стрелки)
   - Text (Текст)
   - Fit (Подгонка)
   - Primary Units (Основные единицы)
   - Alternate Units (Альтернативные единицы)
   - Tolerance (Допуски)

## Текст

### Однострочный текст
- **Команда:** TEXT или DT
- **Параметры:** Высота, Угол наклона, Шрифт

### Многострочный текст
- **Команда:** MTEXT или T
- **Преимущества:**
  - Форматирование внутри текста
  - Колонки
  - Списки
  - Символы и специальные знаки
  - Поиск и замена

### Стили текста
- **Команда:** STYLE
- **Настройка:** Шрифт, Высота, Ширина, Наклон

### Специальные символы
| Код | Символ |
|-----|--------|
| %%d | Градус (°) |
| %%p | Плюс-минус (±) |
| %%c | Диаметр (Ø) |
| %%u | Подчёркивание |
| %%o | Надчёркивание |

## Штриховки

### Создание штриховки
- **Команда:** HATCH или H
- **Типы:** Сплошная, Линейная, Крестовая, Узорная
- **Параметры:** Масштаб, Угл, Точность

### Паттерны штриховок
- ANSI31 (металл)
- ANSI32 (сталь)
- ANSI37 (земля)
- Сплошная (SOLID)
- Много другие

## Таблицы

### Создание таблицы
- **Команда:** TABLE
- **Источник:** Блоки атрибутов, CSV, Excel
- **Формат:** Строки, Столбцы, Ячейки

## Линейные стили

### Типы линий
| Тип | Описание |
|-----|----------|
| Continuous | Сплошная |
| Dashed | Штриховая |
| Center | Центровая |
| Hidden | Невидимая |
| Phantom | Фантомная |
| Dashdot | Штрихпунктирная |

### Загрузка линий
- Команда LINETYPE
- Из файлов .lin
- Создание пользовательских

## Масштабирование аннотаций

### Аннотативные объекты
- Текст, Размеры, Сноски, Блоки
- Масштаб привязан к листу
- Автоматическое масштабирование

### Масштабы чертежа
- **Команда:** SCALELISTEDIT
- Стандартные: 1:1, 1:2, 1:5, 1:10, 1:100
- Пользовательские масштабы
