# nanoCAD 26 - Команды 2D черчения

## Источник
https://nanocad.com/learning/online-help/nanocad-platform/geometric-objects/

## Линейные объекты

### Линия (LINE)
- **Команда:** LINE или L
- **Описание:** Отрезок по двум точкам
- **Параметры:** Начальная точка → Конечная точка
- **Привязки:** Конец, Середина, Пересечение, Примыкание

### Полилиния (PLINE)
- **Команда:** PLINE или PL
- **Описание:** Связанная последовательность отрезков и дуг
- **Преимущества:** Единый объект, разная ширина, замкнутость
- **Параметры:** Ширина начала/конца, Дуга, Замыкание

### Многоугольник (POLYGON)
- **Команда:** POLYGON
- **Описание:** Правильный многоугольник
- **Варианты:** Вписанный/описанный, по числу сторон

### Прямоугольник (RECTANG)
- **Команда:** RECTANG или REC
- **Описание:** Прямоугольник
- **Способы:** По 2 точкам, по 3 точкам, по центру

### Многолиния (MLINE)
- **Команда:** MLINE
- **Описание:** Параллельные линии (стены, трубы)
- **Настройка:** Стили многолинии

## Криволинейные объекты

### Окружность (CIRCLE)
- **Команда:** CIRCLE или C
- **Способы построения:**
  - По центру и радиусу
  - По центру и диаметру
  - По 3 точкам
  - По 3 касательным
  - По 2 касательным и радиусу

### Дуга (ARC)
- **Команда:** ARC или A
- **Способы построения (11 вариантов):**
  - По 3 точкам
  - По началу, центру, концу
  - По началу, центру, углу
  - По началу, концу, углу
  - По началу, концу, радиусу
  - И другие комбинации

### Эллипс (ELLIPSE)
- **Команда:** ELLIPSE
- **Параметры:** Центр, конец оси, коэффициент сжатия

### Сплайн (SPLINE)
- **Команда:** SPLINE
- **Описание:** Гладкая кривая NURBS
- **Способы:** По точкам на кривой, по контрольным точкам

### Кольцо (DONUT)
- **Команда:** DONUT
- **Описание:** Два концентрических окружности (заполненные)

### Спираль (HELIX)
- **Команда:** HELIX
- **Описание:** 3D-спираль (основа для резьбы, пружин)

## Вспомогательные объекты

### Точка (POINT)
- **Команда:** POINT или PO
- **Стили:** Точка, крестик, круг, линия

### Бесконечные линии (XLINE, RAY)
- **XLINE:** Двойная бесконечная линия
- **RAY:** Луч (полубесконечная)
