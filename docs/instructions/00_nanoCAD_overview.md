## Модульная конфигурация

nanoCAD 26 строится по модульному принципу:

| Модуль | Назначение | Цена (год) |
|--------|-----------|------------|
| **Platform** | Базовая 2D/3D платформа, DWG-совместимость | от $289 |
| **3D Solid Modeling** | Параметрическое и прямое 3D-моделирование | модуль |
| **Mechanica** | Механическое проектирование, библиотека стандартных изделий | модуль |
| **Construction** | Строительное проектирование, IFC | модуль |
| **Raster** | Работа с растровыми изображениями | модуль |
| **Topoplan** | Цифровое моделирование рельефа (TIN) | модуль |
| **3DScan** | Обработка облаков точек | от $1499 |
| **Free** | Бесплатная версия (ограниченная) | бесплатно |

## Системные требования

- **ОС:** Windows 10/11 (64-bit)
- **Процессор:** Intel Core i3 или выше
- **RAM:** от 8 ГБ (рекомендуется 16+ ГБ)
- **Диск:** от 1 ГБ свободного места
- **Видеокарта:** DirectX 10 совместимая
- **Разрешение:** 1920x1080 или выше

## Форматы файлов

| Формат | Описание |
|--------|----------|
| **DWG** | Основной формат (native) |
| **DXF** | Обменный формат |
| **DWT** | Шаблоны чертежей |
| **STEP/IGES/STL** | 3D-обмен (через модуль 3D Solid Modeling) |
| **PDF** | Импорт/экспорт |
| **IFC** | BIM-обмен (через модуль Construction) |

## API и автоматизация

nanoCAD поддерживает несколько API для разработки плагинов:

| API | Язык | Назначение |
|-----|------|-----------|
| **.NET** | C#, VB.NET | Плагины, команды, UI |
| **C++** | Native C++ | Высокопроизводительные плагины |
| **LISP** | AutoLISP/Visual LISP | Скрипты, автоматизация |
| **COM** | VBScript, Python, JavaScript | Автоматизация через COM |
| **JavaScript** | Встроенный JS | Макросы |

## Ключевые возможности платформы

### 2D Черчение
- Полный набор примитивов (линии, дуги, окружности, полилинии, сплайны, эллипсы)
- Работа со слоями, линиями, цветами
- Размерные линии и.annotate
- Блоки и внешние ссылки (XREF)
- Штриховки и заливки
- Печать и экспорт в PDF

### 3D Моделирование (модуль 3D Solid Modeling)
- Параметрическое моделирование (2D эскизы → 3D тела)
- Прямое редактирование 3D (push/pull)
- 3D-ограничения (Mate, Insert, Angle, Tangent, Symmetry)
- Булевы операции (Union, Subtract, Intersect)
- Выдавливание (Extrude), вращение (Revolve), sweep, loft
- Скругления и фаски
- Сборки с историей построения
- Листовой металл

### Механика (модуль Mechanica)
- Библиотека стандартных изделий (ГОСТ, ISO)
- Параметрические калькуляторы
- Спецификации и ведомости
- Автоматизация чертежей механических изделий

### Строительство (модуль Construction)
- Архитектурные инструменты
- Структурное проектирование
- IFC-поддержка (BIM)
- Параметрические библиотеки строительных элементов

## Интерфейс

- **Лента (Ribbon)** - основной интерфейс команд
- **Строка команд** - ввод команд текстом
- **Панель свойств** - редактирование свойств объектов
- **Панель слоёв** - управление слоями
- **Строка состояния** - режимы привязки, отображения
- **Проводник чертежа** - управление структурой DWG
