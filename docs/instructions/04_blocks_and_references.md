# nanoCAD 26 - Блоки и внешние ссылки

## Блоки

### Создание блока
- **Команда:** BLOCK или B
- **Параметры:**
  - Имя блока
  - Точка вставки (базовая точка)
  - Объекты для включения
  - Поведение (Annotative, Scale uniformly)

### Вставка блока
- **Команда:** INSERT или I
- **Параметры:**
  - Имя блока или DWG-файл
  - Позиция вставки
  - Масштаб (X, Y, Z)
  - Угол поворота
  - Разрежение (Explode)

### Библиотеки блоков
- **Команда:** DESIGNCENTER
- **Источники:** DWG-файлы, Палитра инструментов

### Блоки с атрибутами
- **Команда:** ATTDEF (создание), ATTEDIT (редактирование)
- **Применение:** Спецификации, Штампы, Позиции

## Внешние ссылки (XREF)

### Подключение XREF
- **Команда:** XREF или XR
- **Типы:**
  - Attach (Вложение) — зависимая ссылка
  - Overlay (Наложение) — независимая ссылка

### Управление XREF
| Действие | Команда |
|----------|---------|
| Подключить | XREF → Attach |
| Отключить | XREF → Detach |
| Обновить | XREF → Reload |
| Разорвать | XREF → Unload |
| Разрешить | XREF → Bind |

### Преимущества XREF
- Совместная работа над одним чертежом
- Автоматическое обновление при изменении
- Уменьшение размера файла
- Централизованное управление

## Динамические блоки

### Создание динамического блока
1. Создать обычный блок
2. Открыть в редакторе блоков
3. Добавить параметры:
   - Linear (линейный)
   - Polar (полярный)
   - XY (по осям)
   - Rotation (поворот)
   - Flip (отражение)
   - Visibility (видимость)
4. Добавить действия:
   - Stretch (растяжение)
   - Move (перемещение)
   - Scale (масштаб)
   - Rotate (поворот)
   - Flip (отражение)
   - Array (массив)

### Примеры динамических блоков
- Окно с изменяемыми размерами
- Дверь с разными типами открывания
- Электрический символ с настройками
- Мебель с размерными вариантами

## Работа с атрибутами

### Экспорт атрибутов
- **Команда:** ATTEXT
- **Форматы:** DXF, CDF, SDF

### Импорт данных
- Связывание с Excel
- Экспорт в CSV

## Палитра инструментов

### Создание палитры
1. **Команда:** TOOLPALETTES
2. Перетащить блоки на палитру
3. Настроить свойства
4. Сохранить набор палитр

### Использование
- Быстрая вставка блоков
- Настройка свойств при вставке
- Организация по категориям
